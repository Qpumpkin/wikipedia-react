# wikipedia-react

[demo](http://wikipedia.sanvvv.me/)

基于 React 实现的维基百科应用。

由于直接使用了维基百科的 API，可能需要网络工具才能获取到条目。

## 技术栈

React 全家桶：react + redux + react-router 4.0
